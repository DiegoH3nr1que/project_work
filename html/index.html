HTML e CSS

  Na parte de linguagem de programação, estudamos sobre HTML, CSS e JavaScript. Linguagens que apesar de serem consideradas simples, foram as primeiras que eu aprendi e gostei bastante. Sobre HTML vimos sobre a estrutura básica dela, que seria o body, head e o html. As tags e os elementos, que seria o p, h1, img, utilizados para criar um parágrafo, uma escrita com tamanho maior, ou para colocar uma imagem. Existem também os atributos, a com um href, sendo utilizado para referenciar algo, como uma âncora. Também aprendemos sobre como modificar um texto,como por exemplo, a tag b, que seria para deixar um texto em negrito, i para deixar em itálico, u para sublinhado, os cabeçalhos de h1 a h6, quebras de linha br, listas ordenadas ol e listas não ordenadas ul, entre outros. Existem também as tags para tabelas, tr e td. Tr definindo uma linha e td definindo uma célula, além de termos também outros atributos como colspan e rowspan para mesclar células. Vimos também sobre os formulários, form, input, textarea e select, utilizados muito para fazer aquelas partes onde você coloca seus dados num site. Além da criação de div`s, que seriam basicamente uma forma de dividir / criar um container com os dados que você quer dentro dela, para poder estilizar o seu site com css.


  Isto foi resumidamente tudo o que vimos sobre as tags que podemos utilizar no nosso html, mas o que me ajudou mesmo a entender e deixar com que o html ficasse mais organizado e de fácil construção de um projeto foi o HTML Semântico. O html semântico que é uma forma estruturada de criar o seu documento, deixando com que seu programa fique mais organizado. Dentre as tags apropriadas para utilizar em seu projeto com html semântico, são eles: header, que representa o cabeçalho ou uma sessão de início. Nav, que seria a barra de navegação do seu site. Article, que é usado para definir um conteúdo independente e autocontido em um projeto. Section, que é utilizada para dividir o conteúdo em partes distintas e significantes, com base em um tema específico. Aside, que seria uma barra lateral do site. A Além da tag main, que seria o contéudo principal da página, e o footer, que seria o rodapé da página.


   Agora sobre CSS, vimos que a sua função é a estilização do seu site, para deixar com que fique organizado e esteticamente bonito. Primeiro, é preciso fazer um link no html para o css, a partir disso, você poderá modificar toda sua página no html utilizando as div`s criadas. Portanto, é necessário saber qual é o tipo de seletor que você colocou na sua div, se ela foi feita como uma div class ou como uma div id. Logo após, já pode-se começar a codificação, e existem diversos atributos que podemos utilizar em uma div, como por exemplo o color, para definir a cor do texto da div escolhida. Font-family, para especificar qual é a fonte que seria utilizada na div. Border-radius, para deixar a div com as suas bordas curvadas ou até mesmo deixar uma imagem circular, entre outros atributos mais comuns do css. Além disso, pude entender a parte de responsividade, que seria a capacidade do meu site em se adaptar e se exibir de maneira adequada em diversos dispositivos e tamanhos de tela. Para que isso possa acontecer, uma boa dica é utilizar as unidades de medidas flexíveis, como por exemplo, usar porcentagem na largura e altura de uma div, isso fará com que a mudança do tamanho da tela não influência o tamanho da div.


Java Script

  Agora sobre Javascript, aprendi sobre a sintaxe básica e seus elementos essenciais para escrever um código funcional, dentre os elementos essenciais para a sintaxe básica, são elas: Os comentários, a declaração de variáveis (var, let, const), os tipos de dados (string, number, boolean, null e undefined), aprendemos também sobre todos os operadores no Javascript (+, -, *, /, %, >, ===, entre outros) e as estruturas de controle ( if-else, for, do-while, while e switch case), cada um com a sua função: if-else é utilizado para fazer condições, for utilizado para um laço de repetição, do-while com a mesma função que o while, mas sendo obrigado a executar pelo menos uma vez, já o while ele avalia se a condição seja verdadeira, para depois executa-la.

  Além disso, aprendemos também sobre funções em Js, na questão de como declarar uma função, de como chamá-la e como retornar o valor dessa função. Utilizamos bastante as funções na lista de exercícios que fizemos, criamos diversas funções para fazer cálculos de questões. Primeiro, foi necessário pegar os valores do cálculo com um input, após isso, dentro da função é necessário declarar uma variável com o método document.getElement, puxando os valores dentro desse input. Após isso, cria uma outra variável para exibir os valores.

  Dentro do JavaScript também aprendemos sobre DOM (Document Object Model), uma representação em forma de árvore de um documento HTML ou XML, que permite a manipulação dos elementos desse documento por meio de scripts ou programas. É uma interface de programação fornecida pelos navegadores web para permitir que os desenvolvedores acessem, modifiquem e interajam com elementos da página. Basicamente, com o conhecimento do DOM é possível que possa fazer uma interação de um botão para que ele faça um cálculo, para que quando o botão seja apertado, apareça uma div na tela ou até mesmo para que um botão apague todas as escritas de um campo, entre outras diversas coisas que se pode fazer com o DOM.


Banco de Dados

  Na parte de Banco de Dados, vimos que o BD é uma estrutura física, onde as informações são mantidas por um período de tempo incerto, e são utilizados sistemas para gerenciar um banco. Existem vários bancos de dados, sendo um dos mais comuns o banco de dados relacional, este tipo de banco as informações são organizadas em tabelas compostas por linhas e colunas. Cada tabela representa uma entidade, como por exemplo, clientes, produtos ou pedidos, e as relações entre essas entidades são definidas por meio de chaves primárias e chaves estrangeiras. Porém, antes de começarmos a programar definitivamente em SQL, estudamos sobre a modelagem de dados. Dentro da modelagem de dados vimos sobre o diagrama MER (Modelo de entidade relacionamento) que é um modelo conceitual que descreve os objetos, suas características e como se relacionam. Estudamos também o diagrama DER (Diagrama entidade relacionamento) que é uma representação gráfica que mostra as entidades, os atributos e o relacionamento entre as entidades.

  Além disso, aprendemos sobre a simbologia de cada parte existente no diagrama DER e a sua característica. Sobre a entidade, existem dois tipos, a forte e a fraca, sendo designadas pela existência de uma chave primária (entidade forte), existindo independente de outra entidade. Já a entidade fraca, que é dependente de outra entidade para existir e não tem chave primária. Na questão da cardinalidade, que seria como as entidades se relacionam, vimos os três tipos de cardinalidade: (1: 1; 1: N; N: M). E sobre os relacionamentos forte e fraco, que são definidos pela cardinalidade.

  Estudamos também sobre os SGBD´s (Sistemas de gerenciamento de dados), que é um software que permite criar, manipular, e administrar um banco de dados. Basicamente, um SGBD oferece uma interface entre os usuários e o Banco de Dados, facilitando a criação, a recuperação, a atualização, e a exclusão de dados de forma mais eficiente e segura. Alguns desses SGBD´s são: Oracle, mySQL, SQLServer, PostgreSQL, etc.


Automação Industrial

  A respeito do conteúdo de Automação Industrial, estudamos primeiramente sobre como funciona um computador, uma memória RAM, uma memória interna, alguns sensores eletrônicos, como por exemplo: O sensor Ultrassônico. Também, aprendemos sobre como funciona o código binário e a sua conversão e tradução, e um pouco sobre a linguagem C (Sintaxe e funcionalidades) e como ela funciona em um sistema embarcado (Arduino) fazendo montagens com o próprio Arduino. Junto aos sistemas embarcados, aprendemos sobre portas lógicas, como elas funcionam, como são utilizadas e alguns dos tipos de portas lógicas, como por exemplo: AND, OR, NOT, NAND, NOR e suas respectivas tabelas verdade.

  Durante o período de aprendizagem de Automação, foi visto sobre a robótica e sua história, entre os pontos vistos há diversos tipos de robôs, diversos tipos de linguagem de programação para os mesmos e também suas aplicações para cada tipo, alguns exemplos de robôs que foram apresentados para nós foram: Robôs moveis (AGV, Humanoides, robôs fixos), Robôs de estrutura cinemática (Eixos em série e eixos em paralelo), Robôs geométricos (Robô de coordenada cartesiana, coordenada cilíndrica e coordenada esférica). Vimos também sobre cada parte de um robô industrial e os tipos de acionamento. Além disso, vimos também sobre os tipos de ferramentas que podem ser colocadas nos robôs, dentre elas: Eletrolíticos de solda, garra, desbaste, câmera, pistola de tinta, ventosa, garra versável, dentre outros.
